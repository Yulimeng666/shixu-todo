# 事序 · shixu-todo

一个本地优先的个人待办网站，聚焦日程记录、任务分类和高频待办管理。

## 项目概述

`shixu-todo` 是一个基于 React + TypeScript + Vite 构建的个人待办应用，面向单人使用场景。第一版重点解决三个问题：

- 快速记录和整理当天及未来任务
- 用全局分类统一管理个人事项
- 在纯前端架构下完成本地持久化

当前版本不依赖后端服务，不接入登录、云同步或团队协作，打开即可使用。

## 功能特性

### 任务管理

- 新增任务
- 编辑任务
- 删除任务
- 标记完成 / 取消完成

### 任务字段

- 标题
- 备注
- 开始日期
- 截止日期
- 优先级
- 分类

### 视图与筛选

- 今天
- 未来
- 全部
- 已完成
- 搜索
- 按优先级筛选
- 按日期筛选
- 按完成状态筛选
- 按分类筛选
- 按截止日期 / 优先级 / 创建时间排序

### 全局分类体系

预置分类：

- 默认
- 工作
- 生活
- 学习
- 购物

分类规则：

- 左侧分类栏是分类主入口
- 支持新增分类
- 支持删除非默认分类
- 删除分类后，相关任务自动回收到 `默认`
- 未选择分类的任务自动归入 `默认`

### 本地优先

- 使用 IndexedDB 保存任务和分类
- 刷新页面后数据仍然保留
- 无需后端即可完成第一版使用流程

## 数据方案

项目采用本地优先的数据架构：

- Zustand 负责业务状态管理
- Dexie.js 负责 IndexedDB 访问封装
- `todos` 表保存任务数据
- `categories` 表保存全局分类数据
- 应用启动时自动初始化预置分类，并兼容历史任务分类迁移

## 业务规则

- 标题必填，保存前会自动去除首尾空格
- 优先级默认值为 `medium`
- 未选择分类的任务自动归入 `默认`
- 今天视图中新建任务时，如果未填写截止日期，系统会自动补成今天
- 今天和未来视图的判断仍然基于 `dueDate`
- 开始日期用于录入、存储和展示，不参与主视图筛选
